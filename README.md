# Библиотека программ PolarDB

Библиотека предназначена для создания эффективных и специализированных баз данных в среде программирования .NET

Библиотека состоит из основного раздела Polar.DB и специальных расширений Polar.PagedStreams, Polar.Datanode. 
Также имеется целостный проект Factograph.Data. Все части оформлены как .NET Core проекты (*.csproj)

Ряд устаревших или не используемых проектов и фрагментов "спрятаны" в директорию oldandunused.

Документация разделена на Примеры, Описание, Спецификация.
